# Tic-Tac-Toe Game

## 🎮 Project Overview

A classic **Tic-Tac-Toe** game developed in Python as part of the ITCS440 course. This project demonstrates fundamental programming concepts through an interactive command-line game where two players take turns marking X and O on a 3x3 grid.

## ✨ Features

- **Two-player gameplay** – Play with a friend or with an AI on the same device.
- **Input validation** – Ensures players select valid and unoccupied positions
- **Win detection** – Automatically detects winning combinations (rows, columns, diagonals)
- **Draw detection** – Recognizes when the game ends in a tie
- **Clear game flow** – Alternating turns with visual board updates after each move
